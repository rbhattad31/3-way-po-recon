How tolerances work

Each reconciliation run uses the default config profile. The engine applies a 3-tier tolerance system:

  1. Strict tolerance (Qty/Price/Amount %) — within this = MATCHED, no agents needed.
  2. Auto-close band (wider Qty/Price/Amount %) — beyond strict but within this = AUTO-CLOSED, no AI agents fired.
  3. Beyond auto-close — full AI agent pipeline runs (analysis, routing, case summary).
